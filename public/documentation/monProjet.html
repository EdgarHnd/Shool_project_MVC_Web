Mon Projet

1)Liste des récoltes

Un affichage de la table des récoltes de la base de données en conservant le format de la base de données

2)Créer une récolte

Un formulaire qui permet de créer une nouvelle récolte. Je récupére dans les tables producteur et vin les id nécessaires à la création de la récolte.

3)Supprimer une récolte

Un formulaire qui permet de supprimer une récolte éxistante. Je récupére dans les tables producteur et vin les id nécessaires à la suppréssion de la récolte

4)Meilleurs années de récolte

Un tableau qui liste les meilleurs années de récolte selon cette base de données. Cela permet d'utiliser la relation entre la table des vin et et la table des récoltes en regroupant les récoltes par années.

5)Récoltes par producteurs

Un affichage un plus complexe des informations données par la table des récoltes. Cela permet de rassembler les données des trois tables sous un même affichage. Je récupère les producteurs qui sont aussi présent dans la table des récoltes et j'effectue pour chacun d'eu une requette dans la table des récoltes qui permet d'afficher les données de vin et de récolte pour ce producteur.
